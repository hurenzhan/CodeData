# titleBar

> 标题栏组件


## props

属性 | 默认值 | 说明
--- | --- | ---
title | '' String | 标题名
bread | [] Array | 面包屑
showBack | false Boolean | 是否显示返回按钮(电子报表版本管理页面需求,在新建电子报表和修改电子报表指标时可以直接返回版本管理页面)

## event

> nothing
